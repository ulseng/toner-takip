================================================================
GİT OLMADAN MANUEL GÜNCELLEME REHBERİ (PROGRAMSIZ)
================================================================

Bilgisayarınızda Git yüklü değilse, güncellemeyi doğrudan 
GitHub web sitesi üzerinden "Sürükle-Bırak" yöntemiyle yapacağız.

----------------------------------------------------------------
1. ADIM: DOSYALARI HAZIRLA
----------------------------------------------------------------
1.  Bu sayfadan (AI Studio) projenin son halini indirin (Sağ üstteki indirme butonu).
2.  İnen ZIP dosyasını masaüstüne bir klasöre çıkartın.

----------------------------------------------------------------
2. ADIM: GITHUB'A YÜKLEME (Sürükle-Bırak Yöntemi)
----------------------------------------------------------------
Telefonunuzun güncellemeyi alması için kodları GitHub'a atmamız lazım.

1.  www.github.com adresine gidin ve giriş yapın.
2.  Sol taraftaki listeden veya profilinizden "toner-takip" projesine tıklayın.
3.  Dosyaların listelendiği ekranın sağ üst kısmında "Add file" butonuna basın.
4.  Açılan menüden "Upload files" seçeneğini seçin.
5.  Masaüstüne çıkarttığınız proje klasörünün içindeki TÜM DOSYALARI seçin.
6.  Seçtiğiniz dosyaları tarayıcıdaki gri alana sürükleyip bırakın.
7.  Yükleme çubuklarının dolmasını bekleyin.
8.  Sayfanın en altına inin. "Commit changes" başlığı altına "Guncelleme" yazın.
9.  Yeşil renkli "Commit changes" butonuna basın.

Bunu yaptıktan sonra GitHub dosyaları işleyecek ve Vercel otomatik olarak
güncellemeyi algılayıp sitenizi yenileyecektir.

----------------------------------------------------------------
3. ADIM: TELEFONDA KONTROL
----------------------------------------------------------------
1.  Bu işlemden yaklaşık 1-2 dakika sonra telefonunuzdan uygulamayı açın.
2.  Sayfayı yenileyin.
3.  Yeni ikonları veya özellikleri görüyorsanız işlem tamamdır.

----------------------------------------------------------------
HATA ALIYORSANIZ (KAYIT YAPAMIYORSANIZ)
----------------------------------------------------------------
Eğer telefondan hala "Kayıt Eklenemedi" hatası alıyorsanız veritabanı
izinlerini açmanız gerekir:

1.  https://console.firebase.google.com/ adresine girin.
2.  Projenizi seçin -> Sol menüden "Build" -> "Firestore Database".
3.  Üstten "Rules" sekmesine gelin.
4.  Kodların içinde "allow read, write: if true;" yazdığından emin olun.
5.  Değilse düzeltip "Publish" butonuna basın.
